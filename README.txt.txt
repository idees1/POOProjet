ENTER : end the turn

TAB : hide/display healthbar

OUT OF COMBAT :

ARROW KEYS: move 

IN COMBAT :

MOUSE-LEFT-CLICK:

GREEN TILES : possible movement
RED TILES : possible target
YELLOW TILES : possible target for buff
